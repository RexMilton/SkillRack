ProgramID- 14438

Matrix - Sort Odd Integers
The program must accept an integer matrix of size R*C. Then the program must sort the odd integers in the matrix across all rows. 
Finally, the program must print the revised matrix.

Boundary Condition(s):
2 <= R, C <= 50
1 <= Matrix element value <= 10^4

Input Format:
The first line contains R and C separated by a space.
The next R lines, each contains C integer values separated by a space.

Output Format:
The first R lines contain the revised matrix.

Example Input/Output 1:
Input:
3 4
90 5 85 29
53 9 17 88
16 99 41 21

Output:
90 5 9 17
21 29 41 88
16 53 85 99

Explanation:
Here R=3 and C=4.
The odd integers in the given 3*4 matrix are highlighted below.
90 5 85 29
53 9 17 88
16 99 41 21
After sorting the odd integers in the given matrix, the matrix becomes
90 5 9 17
21 29 41 88
16 53 85 99
