Pattern Printing Repeated N (Id-6313)

The program must accept a string value S and an integer N as input. The program must print the pattern as shown in the Example 
Input/Output section.

Boundary Condition:
1 <= Length of S <= 1000
1 <= N <= 999

Input Format:
The first line contains the values of S and N separated by a space.

Output Format:
The first N lines contain the pattern as shown in the Example Input/Output section.

monopoly 4
